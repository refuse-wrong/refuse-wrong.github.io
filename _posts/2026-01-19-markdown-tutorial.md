# 标题 (Headings)
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
```
# 字体格式 (Text Styling)
```
加粗：使用 **文本** 或 __文本__
斜体：使用 *文本* 或 _文本_
斜体加粗：使用 ***文本***
删除线：使用 ~~文本~~
```
# 列表 (Lists)
**无序标题**

使用 *、+ 或 - 加上空格。
```
* 项目一
- 项目二
+ 子项目（缩进两个空格）
```
**有序标题**

使用数字加上 . 和空格。
```
1. 第一步
2. 第二步
3. 第三步
```
# 链接与图片 (Links & Images)
**链接**
```
语法：[显示文本](链接地址 "可选标题")
```
**图片**
```
语法：![替代文字](图片地址 "可选标题")
```
> 注意：图片语法仅仅比链接多了一个感叹号 !。

# 引用与分割线 (Quotes & Rules)
**引用**

使用 > 符号。
```
>这是一个引用示例。
>>也可以嵌套引用。
```
**分割线**
使用三个或更多的 *、- 或 _。
```
---
***
___
```
# 代码 (Code)
**行内代码**

使用反引号 ` 包裹。 
```
`printf()`
```
**代码块**

使用三个反引号 ``` 并可指定语言名称进行高亮。

# 表格 (Tables)
使用 | 分隔单元格，使用 - 分隔表头。通过 : 控制对齐方式。
```
|表头1|表头2|表头3|
|:---|:---:|---:| 
|左对齐|居中对齐|右对齐|
```

# 数学公式 (Mathematics)
对于科学公式，通常使用 LaTeX 语法。

**行内公式**：
```
$E = mc^2$
```

**块级公式**：
```
$$\sum_{i=1}^{n} i = \frac{n(n+1)}{2}$$
```
